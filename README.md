# Dotfiles

[Setup dotfiles git bare repo](https://developer.atlassian.com/blog/2016/02/best-way-to-store-dotfiles-git-bare-repo/)

- Install Xcode from the App Store
- Install macOS Command Line Tool, `xcode-select --install`
- Copy public and private SSH keys to ~/.ssh
  - `chmod 700 $HOME/.ssh`
  - `chmod 600 ~/.ssh/id_rsa; chmod 600 ~/.ssh/id_rsa.pub`
- Clone this repo, follow these: [instructions](https://developer.atlassian.com/blog/2016/02/best-way-to-store-dotfiles-git-bare-repo/)
